Cardify.js

Efecto hover que muestra el atributo alt



Dado un contenedor busca todas las imágenes que encuentre dentro y las reemplaza por un nuevo elemento <figure> que contenga la etiqueta <img> además de un <figcaption> con el texto que tiene el atributo alt de esa imagen.


ENLAZAR

Enlazar el script que vincula el archivo js .

<script src="js/jquery.cardify.js"></script>

Enlazar el link que vincula el archivo css .

<script src="js/jquery.cardify.js"></script>

INVOCAR

Despues de enlazar los archivos, invocar al plugin
Cardify


$(document).ready(() function {
          $('.container').cardify(); 
        })
